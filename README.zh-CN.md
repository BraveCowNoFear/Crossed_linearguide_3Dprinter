# 交叉线轨 3D 打印机

[English](./README.md) | [简体中文](./README.zh-CN.md)

这是一个受 Ultimaker 2 线轨结构启发、并进一步改造成四电机驱动方案的 3D 打印机项目，重点在于更高的结构刚性、更强的驱动力，以及对工程材料的更好兼容性。

## 项目概览

这个仓库主要分享一套自定义打印机的机械设计文件，核心思路包括：

- 使用交叉线轨结构，提高刚性和定位精度。
- 使用四个独立电机，获得更强的运动控制能力。
- 结构件按 PA-CF 等高性能材料使用场景来考虑。
- 开放 CAD 与 STL 文件，方便学习、修改和二次开发。

## 重要说明

这个项目更适合作为结构设计参考，而不是拿来直接无修改量产。尺寸、公差、电气方案、固件和装配细节都需要根据你的实际材料、工艺和需求继续优化。

## 技术规格

| 项目 | 规格 |
| --- | --- |
| 结构类型 | 受 UM2 启发的线轨改进结构 |
| 驱动方式 | 四电机独立驱动 |
| 推荐耗材 | PA-CF，打印温度约 320 C |
| 主板 | 赤兔商品板 |
| 固件 | 主板配套闭源固件 |
| 打印件材料 | 以尼龙碳纤维件为目标设计 |

## 推荐材料

- 耗材：Fusrock PA-CF
- 喷嘴温度：320 C
- 屈服强度参考：190 MPa
- 运动部件：交叉线轨
- 电机：42 级别步进电机
- 控制板：赤兔系列

仓库里没有把所有型号彻底固定死，建议把这里当作起点，再按你自己的 BOM 做适配。

## 仓库内容

- `Picture/`：5 张项目照片
- `Solidworks/`：16 个 SolidWorks 零件文件
- `STL/`：17 个导出的 STL 文件
- `Test model/`：1 个测试模型

其中包含的代表性零件包括：

- X/Y 轴滑块与盖板
- X、Y、Z 轴限位安装件
- 屏幕盒与盖板
- 喷头与吹料风扇相关垫片
- USB 固定件

## 装配流程概览

1. 先打印全部结构件，优先推荐 PA-CF 或其他高强工程材料。
2. 组装 X 轴线轨与滑块总成。
3. 组装 Y 轴结构与平台。
4. 安装 Z 轴运动系统并拼装整机框架。
5. 完成主板、电机、加热器、屏幕和 USB 的接线。
6. 配置固件，并做运动参数和热床调平校准。

## 打印建议

PA-CF 参考参数：

- 喷嘴温度：320 C
- 热床温度：90 到 110 C
- 打印速度：30 到 50 mm/s
- 必须使用硬化喷嘴

PLA 打样参考参数：

- 喷嘴温度：200 到 220 C
- 热床温度：50 到 60 C
- 打印速度：50 到 80 mm/s
